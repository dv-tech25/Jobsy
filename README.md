# Jobsy - Job Application Tracker

Jobsy is a smart and efficient job application tracking web application designed to help users manage and optimize their job search process. With Jobsy, users can add job applications, set deadlines, track statuses, receive reminders before deadlines, analyze progress, and stay organized throughout their career journey.
